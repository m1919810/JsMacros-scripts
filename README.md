# JS脚本仓库
该仓库提供了由matl114开发的运行在jsmacros中的脚本

部分脚本需要依赖[SlimefunHelper](https://github.com/m1919810/SlimefunHelper)运行,点击[**此处**](https://github.com/m1919810/SlimefunHelper/releases)下载最新版本

脚本开头的注释将注明其触发方式,若不存在,则默认为快捷键触发

大部分快捷键触发的脚本可以通过触发第二次快捷键来正常退出,直接中断后果自负

[/libs](./libs/)下的脚本为工具类脚本，会在部分脚本中被导入

有些脚本比较旧,作者会不定期维护, 玩家若是希望理解其用途或者参数如何调整,可以向大语言模型寻求帮助

欢迎各位jsmacros大蛇提建议或者发起pr
